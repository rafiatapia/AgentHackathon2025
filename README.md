# AgentHackathon2025
1. Hackathon.docx file contains the business idea of the hackathon
2. The sample data as well as the functions to generate the sample data is in Data folder
3. The Data folder also contains helper.ts and helper.py function to provide some helper function to speed up the development
4. The Snippet folder contains some jupyter notebook that contains some of the code I showed in earlier sessions
     a. Agent.ipynb shows basic setup of strand agent
     b. DocDB-Vectors.ipynb shows RAG code where the vector DB is AWS DocumentDB
     c. OllamaReference.ipynb A quick reference to some of Ollama cli commands
     d. HuggingFaceBasic.ipynb & HuggingFaceAdv.ipynb shows some use of hugging face transformer api
     
5. There is a complete MCP host/client/server implementation in both typescript and python at https://github.com/rafiatapia/DataGenerator
6. There is sample RAG implementation in Code created by Bhai Murtaza Bhinderwala that can be accessed from https://github.com/murtuzaib/ai_engg
